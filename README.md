# vue-demo

Demo on how to make a _basic_ search engine given during [DotJS 2018](https://2018.dotjs.io).

Includes:

- live querying
- basic typo-tolerance
- basic sorting

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```
